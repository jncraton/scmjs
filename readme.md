A simple [Scheme](<https://en.wikipedia.org/wiki/Scheme_(programming_language)>) interpreter in pure JavaScript

[Playground](https://jncraton.github.io/scmjs/)

## Supported Features

- [ ] Literals
  - [x] Integer
  - [x] Boolean
- [ ] Built-in functions
  - [x] `define`
    - [x] Numeric literals
    - [x] Procedures
  - [x] `display`
  - [x] `newline`
  - [x] `cond`
    - [x] `else`
  - [x] `+`
  - [x] `-`
  - [x] `*`
  - [ ] `/`
  - [x] `=`
  - [x] `>`
  - [x] `<`
